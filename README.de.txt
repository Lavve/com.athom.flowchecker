Erhalten Sie Informationen zu Ihren Homey Flows.
Lassen Sie sich benachrichtigen, wenn sich ein Wert geändert hat.

Aktuelle Funktionen:
- Benachrichtigung über beschädigte Flows (standardmäßig aktiviert)
- Benachrichtigung über deaktivierte Flows (standardmäßig deaktiviert)
- Benachrichtigung über Flows mit beschädigten Variablen (standardmäßig aktiviert)
- Übersicht über beschädigte Flows
- Übersicht über deaktivierte Flows
- Übersicht über Flows mit beschädigten Variablen
