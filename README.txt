Retrieve information from your Homey Flows. 
Get notification when a value has changed.

Current features:
- Notify broken flows (default enabled)
- Notify disabled flows (default disabled)
- Notify flows with broken variables  (default enabled)
- Notify for unused flows
- Notify for unused logic
- Overview broken flows
- Overview disabled flows
- Overview flows with broken variables (logic, devices, apps, Better Logic, Flow Utilities)
- Overview unused flows
- Overview unused logic
- Exclude folders in Flowcheck
- Better Logic & Flow Utilities support
