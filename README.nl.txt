Haal informatie op uit Homey Flows. 
Ontvang een melding wanneer een waarde is gewijzigd.

Huidige functies:
- Notificatie kapotte flows (standaard ingeschakeld)
- Notificatie uitgeschakelde flows (standaard uitgeschakeld)
- Notificatie kapotte variabelen in flows (standaard ingeschakeld)
- Notificatie voor ongebruikte flows
- Notificatie voor ongebruikte logic
- Overzicht kapotte flows
- Overzicht uitgeschakelde flows
- Overzicht kapotte variabelen in flows (logic, apparaten, apps, Better Logic, Flow Utilities)
- Overzicht ongebruikte flows
- Overzicht ongebruikte logic
- Filter mappen in Flowcheck
- Better Logic & Flow Utilities ondersteuning
